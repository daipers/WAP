# Milestone Requirements: v1.0
**Archived:** 2026-02-27
**Status:** Shipped

## v1 Requirements (All Completed)

### Identity & Session
- [x] **IDEN-01**: User can create and manage identity records for assessment candidates (Validated)
- [x] **IDEN-02**: User can authenticate and authorize access to assessment sessions (Validated)
- [x] **IDEN-03**: Session persists across browser refresh and maintains attempt state (Validated)

### Content Bank
- [x] **CONT-01**: User can import items from QTI packages (Validated)
- [x] **CONT-02**: User can export items to QTI format for portability (Validated)
- [x] **CONT-03**: User can store items with metadata (tags, difficulty, time limits) (Validated)
- [x] **CONT-04**: User can version items and track changes (Validated)

### Test Assembly
- [x] **TEST-01**: User can assemble tests from item bank with sections (Validated)
- [x] **TEST-02**: User can define item selection rules and ordering (Validated)
- [x] **TEST-03**: User can set timing, navigation, and attempt limits (Validated)

### Delivery
- [x] **DELV-01**: Candidate can take assessment with timed, navigable interface (Validated)
- [x] **DELV-02**: Candidate can save progress and resume within session (Validated)
- [x] **DELV-03**: Server maintains authoritative timing even on client disconnect (Validated)

### Scoring
- [x] **SCR-01**: System scores response-based items automatically (Validated)
- [x] **SCR-02**: System supports custom scoring rules per item/test (Validated)
- [x] **SCR-03**: Scores are versioned and reproducible (Validated)

### Reporting
- [x] **REPT-01**: System generates candidate scorecard with breakdown (Validated)
- [x] **REPT-02**: System exports results to CSV format (Validated)
- [x] **REPT-03**: Scorecards link scores to evidence (item responses) (Validated)

### Audit & Integrity
- [x] **AUDT-01**: System records immutable audit log for each assessment attempt (Validated)
- [x] **AUDT-02**: System captures delivery events (start, answer, submit, terminate) (Validated)
- [x] **INTG-01**: System supports configurable lockdown/integrity controls (Validated)
- [x] **INTG-02**: Integrity events are logged with timestamps (Validated)

### Integration
- [x] **INTG-03**: System supports LTI 1.3 launch from LMS (Validated)
- [x] **INTG-04**: System sends grade passback via LTI Outcomes (Validated)
