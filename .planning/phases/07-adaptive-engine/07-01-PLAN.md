  <files>dev_package/src/cat_service/irt_engine.py</files>
  <action>
    Create a new service for IRT calculations using numpy/scipy.
    Implement:
    1. calculate_3pl_probability(theta, a, b, c): Returns probability of correct response.
    2. estimate_theta_eap(prior_theta, prior_sd, responses, item_params):
       - Uses Expected A Posteriori (EAP) method.
       - Implements quadrature-based integration over the theta range [-4, 4].
    3. calculate_fisher_information(theta, a, b, c): Returns the information value for a given theta.
    4. select_next_item_mfi(theta, available_items):
       - Iterates through available_items and returns the one that maximizes Fisher Information at the current theta.
  </action>
  <verify>
    Run unit tests with known response patterns (e.g., all correct, all incorrect) and verify that the EAP estimate moves in the expected direction and matches theoretical values.
  </verify>
  <done>

  </done>
